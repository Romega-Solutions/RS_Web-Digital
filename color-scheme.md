# Romega Color Scheme

This file captures the current brand colors from the active Romega mood board.

## Core Colors

| Role | Hex | Usage |
|---|---|---|
| Primary | `#FEBD18` | Main CTAs, highlights, key actions |
| Secondary | `#358DF0` | Secondary actions, links, navigation accents |
| Tertiary | `#A89FF8` | Soft accent blocks, decorative emphasis |
| Accent | `#FD755E` | Warm promotional accents and callouts |
| Neutral | `#424861` | Base dark text, structure, and grounded UI surfaces |
| Surface | `#FFFFFF` | Default page and card surfaces |
